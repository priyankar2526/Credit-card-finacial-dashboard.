# Credit-card-finacial-dashboard.
Overview
This project is an interactive Power BI dashboard designed to analyze credit card Transaction

🎯 Objectives
- Analyze overall revenue and transaction volume  
- Understand customer spending behavior  
- Identify top-performing categories and segments  
- Track trends over time
  
 📊 Key Metrics
- Total Revenue: 55M+  
- Total Transactions: 576M+  
- Average Transaction Value: 7.84M  
- Customer Segmentation based on demographics
  
  📊 Features
- Revenue analysis by expenditure category  
- Customer analysis by age, gender, job, and education  
- Monthly and quarterly revenue trends  
- Category-wise and customer-wise insights  
- Interactive filters and slicers
  
  🛠️ Tools Used
- Power BI  
- Microsoft Excel
